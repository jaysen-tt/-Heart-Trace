## 关于 (About)

生命是一张单程票，你将如何度过这一生？
「每日刻度」不仅仅是一个日记本，它是你忠实的情绪伴侣和生命记录者。在这个喧嚣的世界里，我们希望能为你提供一片静谧的角落，让你慢下来，记录每一个真实的情绪瞬间，感知时间的流逝与生命的珍贵。

### 核心功能

*   **情绪光谱追踪**：摒弃枯燥的文字记录，用色彩定义心情。从“超棒”到“崩溃”，15种细腻情绪状态，构建你的专属情绪光谱。
*   **沉浸式日记体验**：极简优雅的编辑器，支持图文混排，记录每日瞬间。内置雨声、炉火、白噪音等多种氛围音乐，让你在书写时瞬间进入心流状态。
*   **生命进度可视化**：独特的“生命倒计时”与“今年余额”小组件，直观感受时间的流逝。

## 隐私政策 (Privacy Policy)

**生效日期：2025年01月01日**

本应用尊重并保护所有使用服务用户的个人隐私权。为了给您提供更准确、更有个性化的服务，本应用会按照本隐私权政策的规定使用和披露您的个人信息。但本应用将以高度的勤勉、审慎义务对待这些信息。

### 1. 信息收集与使用

*   **本地存储**：您的所有日记内容、心情记录、图片等数据均**仅存储在您的本地设备上**。我们不会上传、收集或存储您的任何个人隐私数据到我们的服务器。
*   **通知访问权限 (Android)**：
    *   **用途**：本应用申请“通知使用权”仅为了读取第三方音乐播放器（如网易云音乐、QQ音乐等）的当前播放状态（歌曲名、歌手、封面、进度），以便在您写日记时提供沉浸式的音乐控制体验。
    *   **限制**：我们**仅**读取音乐类应用的通知，绝不会读取短信、聊天软件（如微信）等其他应用的通知内容。
    *   **数据处理**：获取的音乐元数据仅在本地内存中临时使用，**不会**上传至任何服务器，也不会保存到本地文件。功能结束后立即释放。
*   **存储权限**：仅在您主动选择“更换头像”、“添加日记图片”或“保存统计图表”时申请，用于读取或写入您指定的图片文件。

### 2. 信息披露

由于我们不收集您的数据，因此不存在向第三方披露您个人信息的情况。

### 3. 政策更新

本隐私政策可能不时更新。您在继续使用本服务时，即视为您接受本隐私政策的变更。
